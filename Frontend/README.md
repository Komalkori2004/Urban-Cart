# 🛒 UrbanCart

UrbanCart is a production-focused full-stack MERN Ecommerce application built to implement real-world ecommerce architecture and modern web development practices.

The project includes authentication, product management, cart functionality, wishlist system, order processing, category management, and an admin dashboard with role-based access control.

---

# 🚀 Tech Stack

## Frontend

* React.js
* Vite
* Redux Toolkit
* React Router DOM
* Axios
* CSS

## Backend

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* JWT Authentication
* bcryptjs
* Cloudinary

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Komalkori2004/Urban-Cart.git
```

## Backend Setup

```bash
cd Backend
npm install
npm run dev
```

## Frontend Setup

```bash
cd Frontend/UrbanCart
npm install
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file inside the Backend directory.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_URI

JWT_SECRET=YOUR_SECRET_KEY

CLOUDINARY_CLOUD_NAME=YOUR_CLOUD_NAME
CLOUDINARY_API_KEY=YOUR_API_KEY
CLOUDINARY_API_SECRET=YOUR_API_SECRET
```
