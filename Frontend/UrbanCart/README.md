# 🛒 UrbanCart

UrbanCart is a full-stack MERN Ecommerce application built to learn and implement real-world ecommerce architecture using MongoDB, Express.js, React.js, and Node.js.

---

# 🚀 Tech Stack

## Frontend
- React.js
- Vite
- React Router DOM
- Axios
- Redux Toolkit
- CSS

## Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcryptjs
